# Overview
This is where you find the basic configuration settings for your server.
Below is the description of the available configureation options:

| field | details | example |
| :---: | :---: | :---: |
| Server Name | The name of your server in the Web UI | The greatest server |
| Description  | A small optional description to help identify your server | Cool server Description |
| Server Port | The port your minecraft server will be accesable from | 25565 |
| Memory | Configure how many MB of Ram the server can use | 4096 MB |
| Max Players | The maximum number of players that can connect to your server | 20 |
| Minecraft Version  | The version for the game, used for mods, players, and addons | 1.20.1 |
| Mod Loader | What server software to use to run minecraft | Paper, Vanilla, Fabric |
| Docker Image | What java version to run the server with | Java 25 |
| TPS command | The command the DiscoPanel uses to check the current tps of your server. Use ```??``` to make backups | tps, fabric tps ?? |
| Detached Mode | Allow the server to continue running after DiscoPanel Stops |
| Auto Start | Allow the server to auto start on the starting to DiscoPanel |
| Additional Ports | Configure extra ports for mods, plugins, or services | BlueMap 1234 1234 TCP |
