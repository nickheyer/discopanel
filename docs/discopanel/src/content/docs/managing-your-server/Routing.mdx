---
title: Routing
description: Configuring a DNS domain for your server instead of an ip address.
---

In this guide you will learn how to use a domain instead of an IP address to connect to your server

Requirements:
 - A domain name (i.e. example.minecraft.net)
 - Access to your router
    - You need access to the router's management UI, most commonly found at addresses like ```http://192.168.0.1``` or ```http://192.168.1.1```
    - You also need to know the username and password for that UI.

Steps:
 1. Navigate to the web interface of DiscoPanel
 2. Go to Settings > Routing
 3. Enable Proxy Configuration
 4. Paste in your domain name (i.e. example.minecraft.net)
 5. Click Save
 6. Scroll down to Proxy Listeners
 7. Take note of the port shown (```:25565```). You will need to forward this port on your router
 8. Go to the server you wish to route
 9. Click on routing and copy the URL shown and paste that in the box below and click save
 10. Start your server, when it is started it will be accessible from the address shown below Current Routing Status
